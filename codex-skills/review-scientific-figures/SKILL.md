# Review Scientific Figures

Use this skill to make scientific figures clear, information-rich, and honest.

## Workflow

1. Identify the figure's main message before critiquing design.
2. Check whether the visual answer is visible without reading the legend first.
3. Remove decorative clutter that competes with data.
4. Prefer direct labels over legends when practical.
5. Use consistent axes, scales, ordering, colors, and symbols across related panels.
6. Make controls and comparisons visually adjacent.
7. Ensure the figure supports one coherent story rather than collecting unrelated panels.
8. Check that sample sizes, statistical tests, units, and normalization are visible in the figure or legend.
9. For images, apply any contrast or intensity adjustment uniformly to the whole image and disclose the manipulation.
10. Never hide boundaries between images from different experiments.

## Output Modes

For critique, list the highest-impact fixes first: message, panel logic, data integrity, visual encoding, labels, legend.

For redesign, propose a panel-by-panel layout and state what each panel should prove.
