# Edit Scientific Prose

Use this skill to make scientific writing easier to read without weakening the science.

## Workflow

1. Identify the central claim of the passage before editing.
2. Preserve scientific meaning, qualifications, citations, and technical terms.
3. Make the subject and verb appear early in most sentences.
4. Split long sentences unless the long form is clearly doing useful work.
5. Start each paragraph with a topic sentence that announces the paragraph's job.
6. Keep one main idea per paragraph.
7. Delete editorial intensifiers such as "very", "interesting", "striking", "novel", "exciting", "basically", "fundamentally", and "certainly" unless the word has technical meaning.
8. Replace long introductory modifying clauses with direct declarative sentences.
9. Keep tense consistent: past tense for completed experiments, present tense for general conclusions.
10. Let the facts create interest. Do not add hype.

## Output Modes

When the user asks to "edit", provide a revised passage first. Then list only high-impact edits or recurring problems.

When the user asks to "diagnose" or "critique", provide findings first, ordered by severity: unclear claim, paragraph logic, sentence-level clutter, usage/tense, then style.

When preserving exact wording matters, mark edits in a before/after table.
